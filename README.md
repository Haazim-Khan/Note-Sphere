# 📌 NoteSphere

NoteSphere is a modern and responsive note management web application built using **React.js**.  
It allows users to create and delete notes dynamically with a clean and intuitive user interface.

---

## 🚀 Features

- ✍️ Add new notes (Title & Description)
- 🗑️ Delete notes instantly
- ⚡ Real-time UI updates using React state
- 🎯 Controlled components for form handling
- 🔄 Unidirectional data flow
- 💡 Clean and responsive design

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- React Hooks (`useState`)
- Functional Components

---

## 🧠 Concepts Implemented

- Component-Based Architecture
- State Management with `useState`
- Lifting State Up
- Props for Component Communication
- Array Mapping for Dynamic Rendering
- Event Handling in React
